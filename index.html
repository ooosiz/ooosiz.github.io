Тематика проекта

Название проекта

Создание коммерческого программного продукта для количественной оценки объема поражения при диссеминированных формах туберкулеза легких.

Название НИОКР 1-ого года (этапа) реализации проекта Разработка алгоритма оценки объема поражения легких. Разработка прототипа (рабочей модели) программного обеспечения для оценки объема поражения легких на основе разработанного алгоритма
Название НИОКР 2-ого года (этапа) реализации проекта Разработка системы взаимодействия специализированного медико-диагностического программного комплекса с пользователями через интернет
Название НИОКР 3-ого года (этапа) реализации проекта Разработка алгоритмов и программных модулей для диагностики разновидностей диссеминированных поражений легких

Аннотация проекта

Разрабатываемый комплекс предназначен в первую очередь для борьбы с таким опасным и распространенным заболеванием как туберкулез, в частности с его очень трудно диагностируемой формой – диссеминированным поражением легких. Существует несколько десятков разновидностей ДПЛ, методы лечения которых порой диаметрально противоположны. Только правильный и своевременный диагноз обеспечит эффективное лечение. Комплекс предполагает не только интеллектуальную составляющую в виде подсказчика, который помогает уточнять диагноз, основываясь на базе примеров и экспертных мнениях, но и, самое главное, дает количественную оценку объема поражения, что позволяет получить объективный показатель динамики лечения.

Пользователь-больной имеет четкий образ своего недуга (что тоже важно при лечении), пользователь-врач получает мощный, надежный инструмент для быстрой диагностики потока больных. Создаваемое ПО обеспечение позволяет строить трехмерные модели легких пациента на основе данных компьютерной томографии (стандартное, обязательное обследование при подозрении на туберкулез), а также численно измерить такой важный показатель, как объём поражения при диссеминированном туберкулезе, силикозе, саркаидозе и других диссеминациях, уточнить размеры и координаты миниобразований.
